## 🌟 马云体话术 (Emotion-Driven)

### 经典反问型
```
您好，我跟您讲实话，这个价格我觉得还有空间。
为什么？因为同款在XX平台才卖XX元，你们的质量我相信不比他们差，
所以我想问问，能不能给个友情价？哪怕少10块钱，我马上下单，绝不犹豫！
```

### 情感共鸣型
```
我跟您说，我买过这么多家店，你们的服务是真的好。
但实话实说，XX那边便宜了XX元，我心里有点不平衡啊。
我想问一下，作为老客户，能不能申请一个专属优惠？
我相信好的产品值得更好的价格，您说对不对？
```

### 金句植入型
```
我跟您分享一句话：做事要实在，做人要真诚。
我看到同款在XX平台XX元，说实话，这个价格让我有点心动。
但是！我更愿意支持服务好的店铺。
所以我想问一下，您这边能不能给个匹配价？哪怕少一点点，
我都愿意在这里买，因为信任比便宜更重要，您说是不是这个理？
```

### 紧迫制造型
```
我跟您讲，我现在就差这一步了。
同款我看了好几个平台，XX那边便宜XX元，我心里有数。
但说实话，我更信任你们！为什么？因为你们服务好。
所以我就想问一句——今天能给我一个满意的价格吗？
能的话，我十分钟之内下单；不能的话，我可能就要考虑别家了。
不是威胁您，是想让您知道，一个真诚的客户就在您面前！
```

### 换位思考型
```
我跟您交个底：这个价差，让我下单的时候有点纠结。
XX平台便宜了XX元，但我知道，便宜不一定好。
所以我就想问问您——如果这个价格能再降一点，
哪怕只是XX元的优惠，我立刻下单，而且以后都来你家买。
您帮我申请一下，我也帮您冲个销量，大家双赢，好不好？
```

### 豪爽直接型
```
老板！爽快人不说暗话——
同款XX平台卖XX元，你们这价格我感觉还能聊。
这样，您给个实在价，哪怕少10块20块的，
我马上付款，而且以后买这类东西就认准你们了。
行不行？行的话我现在就下单！
```

## 🎯 预设场景话术

### 场景1: 竞品比价型 (P0)
```
您好，我看到同款在XX平台价格更低（¥XX vs ¥XX）。
如果能匹配价格或提供XX元无门槛券，我立刻下单。
```

### 场景2: 老客户专属型 (P1)
```
您好，我是店铺老客户了（已收藏），看到别家更便宜。
但说实话，我更信任你们的服务。
请问有老客户专属优惠吗？哪怕少一点点，我立刻下单！
```

### 场景3: SKU规格型 (P1)
```
您好，我对比了规格：
- [大号6-8钱]：¥XX
- XX平台[中号4-6钱]：¥XX

如果能申请老客户价，我愿意升规格购买！
```

### 场景4: 批量采购型 (P2)
```
老板！爽快人不说暗话——
我需要采购X份，能给个批发价吗？
价格合适我立刻付款，长期合作！
```

### 场景5: 限时紧迫型 (P2)
```
您好，看到XX平台在做活动（更低）。
如果贵店能再优惠X元，我今天立刻下单！
```

### 场景6: 价格保护型 (P3)
```
您好，我X月X日在贵店购买了XX（订单号：XXX）。
今天发现同款降价了¥XX，请问可以退差价吗？
```

## 📝 话术编写原则

1. **真诚第一**: 永远保持友好，不要威胁
2. **制造价值感**: 让卖家觉得服务好是值得溢价的
3. **明确但不强硬**: 告诉卖家你想要什么，但给台阶下
4. **制造紧迫感**: 表明"现在就差这一步了"
5. **承诺回报**: "以后都来你家买"、"介绍朋友"
6. **换位思考**: "您帮我这个忙，我也帮您冲销量"
7. **SKU感知**: 根据规格大小调整议价预期

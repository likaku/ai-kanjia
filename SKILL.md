# 爱砍价 (AI Negotiator)

## 核心能力

1. **价格监控** - 追踪购物车/收藏夹商品价格变化
2. **竞品比价** - 全网同款商品历史最低价抓取
3. **优惠券检测** - 监测店铺隐藏优惠券/老客价
4. **旺旺议价** - 自动向卖家发起证据化议价话术

## 工作流程

### Step 1: 添加监控商品

用户可通过以下方式添加商品到监控列表：
- 发送商品链接
- 发送淘宝/天猫商品标题关键词
- 从购物车/收藏夹选择

### Step 2: 价格情报收集

```python
# 参考 scripts/price_monitor.py
- 抓取商品当前价格
- 搜索同款全网最低价 (通过search_products)
- 检测店铺优惠券 (通过旺旺接口)
- 记录价格历史到 memory/
```

### Step 3: 议价触发条件

| 触发条件 | 优先级 | 议价策略 |
|---------|-------|---------|
| 竞品价格更低 | P0 | 发同款链接，要求匹配 |
| 发现隐藏优惠券 | P1 | 直接询问优惠 |
| 老客户专属价 | P2 | 表明复购意向 |
| 价格上涨前兆 | P3 | 提醒并建议入手 |

### Step 4: 发送议价话术

典型话术模板：
```
"您好，我看到同款在 [XX平台] 价格更低（[链接]），如果能匹配价格或提供 [X] 元无门槛券，我立刻下单。"
```

## 议价话术库

### 场景A: 竞品比价
```
您好，我看到同款在XX平台价格更低，如果能匹配价格或提供10元无门槛券，我立刻下单。
```

### 场景B: 隐藏券检测
```
您好，看到店铺有优惠活动，请问有针对老客户的专属券吗？买2份可以优惠多少？
```

### 场景C: 竞品+叠加优惠
```
您好，同款在XX平台XX元，你们能匹配这个价格吗？如果能再叠加一个无门槛券，我马上下单。
```

### 场景D: 收藏后议价
```
您好，我是店铺老客户了（已收藏），看到别家更便宜，能给个友情价吗？
```

## 输出格式

每次议价后记录到当日memory文件：

```markdown
## [时间] 议价记录
- 商品: [商品名称]
- 原价: ¥XX
- 竞品价: ¥XX
- 议价结果: [成功/失败/待回复]
- 卖家回复: [回复内容]
- 节省: ¥XX
```

## 注意事项

- 每次只针对1-2款商品议价，避免骚扰卖家
- 同一商品48小时内不重复议价
- 议价成功记录到 MEMORY.md 的"议价成功案例"区
- 保持话术自然，避免机器人感过强
